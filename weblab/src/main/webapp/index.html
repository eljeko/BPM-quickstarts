Task list

The Task page contains a list of tasks related to your group (or profile or role).

Start a very simple process

The minimal test you can try. Just starts a process on Business Central and returns the instance ID of the process. You can later check directly on BPM console, using Process Instaces view, the info about the completed process.

Start a process with exeternal interaction

Same as previous process with addition of a call to an external system. The call to the other system is "mocked" by a ReST service deployed into a different container.
